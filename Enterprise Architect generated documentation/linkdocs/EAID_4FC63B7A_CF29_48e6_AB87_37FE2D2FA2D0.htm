ID: ID_07
Учасники: Зареєстрований користувач, Система
Попередні умови: Зареєстрований користувач має бути зареєстрованим у системі
Результат: Зареєстрований користувач має нову комбінацію логін/пароль
Основний сценарій:
  1. 1.
    Система виводить форму для відновлення паролю;
  2. 2.
    зареєстрований користувач вводить заповнює поля форми;
  3. 3.
    система виконує валідацію введених даних;
  4. 4.
    система сповіщає про результат операції.
Виключні ситуації: валідація даних була виконана неуспішно.
